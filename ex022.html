Observação

O at() método é uma nova adição ao JavaScript.

Permite o uso de índices negativos, mas charAt() não.

Agora você pode usar myString.at(-2) em vez de charAt(myString.length-2).

Observação

Todos os métodos de string retornam uma nova string. Eles não modificam a string original.

Disse formalmente:

Strings são imutáveis: Strings não podem ser alteradas, apenas substituídas.
