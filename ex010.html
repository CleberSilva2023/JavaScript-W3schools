Variáveis em JavaScript

Neste exemplo, x, y e z são variáveis.

Nota

O que é var foi utilizada palavra-chave em todo o código JavaScript de 1995 a 2015.

O que é let E a const As palavras-chave foram adicionadas ao JavaScript em 2015.

O que é var a palavra-chave só deve ser usada em código escrito para navegadores mais antigos.

Quando usar var, deixar ou const?

1. Sempre declare variáveis

2. Sempre use const se o valor não deve ser alterado

3. Sempre use const se o tipo não deve ser alterado (Arrays e Objects)

4. Apenas o uso let se você não pode usar const

5. Apenas o uso var se você tem que suportar navegadores antigos.
